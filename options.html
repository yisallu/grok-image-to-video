Grok 图生视频

网页上 Alt + 左键点击 任意图片即可生成视频,在新标签页显示进度并播放结果。
左 Alt + 左键点击图片时使用。留空则用内置默认。保存非空提示词会自动加入历史。
右 Alt + 左键点击图片时使用。留空则回退用提示词 A。
1–15 秒
xAI 仅支持 480p/720p
点多张图时最多同时跑几个,其余自动排队。默认 3;太高易触发 xAI 限流。
默认 grok-imagine-video-1.5-preview;也可改成接口支持的其它视频模型。
连接与推送配置
可填 https://api.5203333.xyz、https://api.5203333.xyz/v1,或旧代理 https://你的域名/i2v。
使用 api.5203333.xyz 时填写它认可的 API Key;旧 /i2v 代理才填写 I2V_SECRET。
直连 api.5203333.xyz 时由扩展推送;旧 /i2v 代理仍由服务端环境变量推送。
填写目标聊天、频道或群组 ID。Bot 需要有发送权限。
留空 = 不归档。生成完成后把视频保存到你自己的网页。
与服务器 VIDEO_ARCHIVE_TOKEN 一致。
